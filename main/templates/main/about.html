Some information about the Game of Life

Developed by John Horton Conway, the game of life is a game that uses cells on a 2D or 3D space to base a set of rules and determine states to create a range of projections.

The states of cells can vary between games and projections but for a base understanding cells would either be alive (turned on / lit) or dead (turned off / unlit) along with a base rule set of the following:

  • Any live cell with fewer than two live neighbours die, as if caused by under population.
  • Any live cell with two or three live neighbours lives on to the next generation.
  • Any live cell with more than three live neighbours dies, as if by overpopulation.
  • Any dead cell with exactly three live neighbours becomes a live cell, as if by reproduction.

Following these and a unique starting state, many have created their own projections and patterns. Try your own variations in the view games tab
